// the first/second/etc. + noun + who/that + verb
// can be transform to: the first/second/etc. + noun + to + verb
// She was the first person to arrive.
// => She was the first person who arrived.
// He’s the last one to leave.
// => He’s the last one who left.
// This is the only student to pass the test.
// => This is the only student who passed the test.
// She is the only person who knows the answer.
// => She is the only person to know the answer.
// This is the best place that we can visit.
// => This is the best place to visit.
